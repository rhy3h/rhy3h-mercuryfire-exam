# 水星防火工程顧問有限公司線上測驗

[Live Demo](https://rhy3h-mercuryfire-exam.vercel.app/)

## 專案設定

```sh
npm install
```

### 開發環境啟動

```sh
npm run dev
```

### 生產環境打包

```sh
npm run build
```

### 環境變數設定

在專案根目錄新增 `.env` 檔案，並填入以下內容：

```env
VITE_API_BASE_URL=https://api-frontend-interview-server.metcfire.com.tw
VITE_INTERVIEWER_NAME=RenHaoYeh
```
